# Sports-Themed Photo Booth (AI Sports Card)

## Summary

Participants take a photo and receive an AI-generated sports trading card.

## K-12 Learning Goals

Learn how AI can transform inputs into creative outputs; intro to APIs and generative tools.

## Undergraduates Learning

API integration, prompt design, user experience, and automated workflow design.

## Overview

Participants can enter booth, take a photo, and have a sports card made for them. Undergraduates learning how to use an API key to make an AI assisted baseball card.

## Booth Details

- **Experience**: Photo capture + AI-generated sports card
- **Technology**: API integration for AI image generation
- **Purpose**: Engaging outreach hook and AI education
- **Setting**: Campus events and outreach activities
- **Duration**: 15 minutes per participant
