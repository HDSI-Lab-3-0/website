# AI Prompt: Event Markdown (events collection)

## Instructions for AI

Generate a complete **MDX or Markdown** document for an **event** in this repo. Output the **entire file** in one fenced code block so it can be copied into `src/content/events/<slug>.mdx` (or `.md`).

Match `src/content.config.ts` (`events` collection). Frontmatter is validated with Zod. Image fields use Astro `image()` — paths must be **relative to the event file** into `src/assets/` (commonly `src/assets/events/`).

---

## Frontmatter

### Required

| Field | Type | Notes |
| --- | --- | --- |
| `eventName` | string | Title in UI |
| `eventDescription` | string | One-line summary for cards / meta |
| `eventDateStart` | datetime | ISO-8601, e.g. `2026-06-15T14:00:00.000Z` |
| `eventDateEnd` | datetime | Same day or multi-day; ≥ start |
| `eventStatus` | enum | `upcoming` \| `ongoing` \| `completed` \| `cancelled` |
| `eventLocation` | string | Venue or online |
| `eventFor` | string | Audience line |
| `eventImage` | image path | **Required** card/default image; path into `src/assets/` |

### Optional

| Field | Type | Notes |
| --- | --- | --- |
| `heroImage` | image path | Used for detail hero if `imageGif` is absent, before `eventImage` |
| `imageGif` | image path | **First** choice for detail hero when set |
| `eventTags` | string[] | Filters on `/events`; align with `src/data/tags.json` when possible |

**Detail hero order:** `imageGif` → `heroImage` → `eventImage`.

**Listing card image order:** `eventImage` → `imageGif` → `heroImage` (then site placeholder).

**Images:** e.g. `../../assets/events/my-cover.webp`. Do **not** point collection `image()` fields at arbitrary `public/` URLs — use `src/assets/` and relative paths. Static fallbacks in the UI may still reference `/assets/...` from `public/`.

## Body structure

After frontmatter:

- Start with `#` + event name and an overview paragraph.
- Sections: schedule, speakers, logistics, registration, contact, etc.
- Use lists, tables, blockquotes, bold for notices.

---

## Example (full file)

```mdx
---
eventName: "Example Summer Workshop"
eventDescription: "One-day hands-on robotics for high school students."
eventDateStart: 2026-07-10T15:00:00.000Z
eventDateEnd: 2026-07-10T21:00:00.000Z
eventStatus: "upcoming"

eventImage: "../../assets/events/example-card.webp"

  - "at UCSD"

# Example Summer Workshop

Opening paragraph: who should attend and what to expect.

## What to bring
```
